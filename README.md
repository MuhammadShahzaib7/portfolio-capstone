# Portfolio Capstone

A personal portfolio website built as a capstone project to showcase projects, skills, experience, and contact information.

## Goals

- Present a polished, recruiter-friendly personal brand
- Demonstrate full-stack web development skills
- Ship a fast, accessible, and responsive single-page or multi-section site
- Deploy to production with a custom domain

## Planned Stack

- **Framework:** Next.js (App Router) with React and TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel (or similar static/edge host)

## Getting Started

```bash
# Install dependencies (once the app is scaffolded)
npm install

# Run the development server
npm run dev

# Build for production
npm run build

# Lint and type-check
npm run lint
npm run typecheck
```

Open [http://localhost:3000](http://localhost:3000) in your browser after starting the dev server.
